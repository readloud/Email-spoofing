# gmailMailSender

This email sender send emails separately from gmail provider to multiple email list.

-To run the application you need to have JRE installed in your OS
-You need to put the email list you want to send to in the file to.txt
-You put the subject of the email in subject.txt
-The body of the email in body.txt (or body.html)
-The list of joined file in the files directory
-Execute the following command line: java -jar emailSend.jar
-Input your login/password

Note:
- There is no validations, you have to follow the project schema or it won't work.
- useEmail@gmail.com and useEmail are both accepted while inserting the login.
- you can use html body by just changing the file extension to .html and write the body in html language
